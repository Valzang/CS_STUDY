- 첫 웨이브는 느리게 움직이고 공격함. 점점 빠르게 움직이고 빠르게 공격.
- 한 웨이브에 총 30마리가 있음. 30마리가 사라지면 다음 웨이브 시작

- 플레이어는 목숨 3개 가지고 시작, 1개는 소모된 상태로 시작되고 실질적으로 우측 바에 2개가 띄워져있음.

- 플레이어는 좌우로만 움직이고, 공격하면 탄을 발사, 맞출시 적은 제거되고 점수는 올라감.

- 플레이어의 목숨이 전부 사라지거나, 적이 화면보다 아래에 도달하면 게임 오버

- 배경의 별들은 반짝반짝 거리지만 게임에 영향을 주지 않음

- 적 ===== star : 10점 / spaceship 20점 / saucers 30점 / bugs 40점 / satellites 50점 (행동양식은 동일)
	- 화면 위쪽에서 시작해서 지그재그 느낌으로 화면 왼쪽에 닿을때까지 이동, 그 후 아랫줄로 한칸 이동후 오른쪽으로 쭉 (반복)
	- 첫 웨이브 몬스터는 한번에 2개씩 발사 가능 (플레이어와 동일), 두 번째는 3개씩.. 3번째는 4개씩...

- 플레이어의 발사체는 2개까지만 발사할 수 있음. 발사체가 화면을 벗어나서 사라지거나 부딪히기 전에는 추가 발사 못함.
- 플레이어가 현재 <- / -> 어느 키를 누르고 있는지 보여줘야함.


- 폼 오브젝트 : 키가 눌렸을 때 적용. 렌더링 시키는 곳. Paint event handler를 가지고 있음. ( Game Object의 Draw() )
- 게임 오브젝트 : 플레이어 목숨이 얼마나 남았는지, 얼마나 많은 웨이브가 있었는지, 게임오버되면 form에게 타이머를 멈추라고 신호를 보냄
	- 리스트<Invader> : 모든 인베이더들은 리스트에 있음. 인베이더가 파괴되면 리스트에서 사라짐.
	- 플레이어 오브젝트 : 좌우 키에 따라 이동. 화면 밖으로 가지 못함
	- 리스트<Shot> : 인베이더를 향한 플레이어의 탄환
	- 리스트<Shot> : 플레이어를 향한 인베이더의 탄환
